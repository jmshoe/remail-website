# Design References

This directory stores design inspiration and reference materials for REmail.

## How to Use

### Adding References

1. **Screenshots**: Save screenshots of designs you like as PNG/JPG files
   - Name them descriptively: `hero-split-layout.png`, `pricing-table-stripe.png`

2. **URLs**: Add website URLs to the `INSPIRATION.md` file below

3. **Use in Commands**: Reference these with `/design-from-reference`
   ```
   /design-from-reference docs/design-references/hero-split-layout.png
   /design-from-reference https://stripe.com/pricing
   ```

## Inspiration Sources

### Recommended SaaS Sites to Study
- **Stripe** - Clean, professional, excellent typography
- **Linear** - Modern, minimal, great animations
- **Vercel** - Developer-focused, dark mode done right
- **Notion** - Friendly, approachable, great illustrations
- **Loom** - Video-forward, engaging CTAs
- **Airtable** - Colorful, organized, great feature pages

### Real Estate Tech Examples
- **PropStream** - Direct competitor, data-focused
- **BatchLeads** - Skip tracing, list building
- **DealMachine** - Mobile-first, driving for dollars
- **REIPro** - Wholesaler focused

## File Naming Convention

Use this naming pattern for screenshots:
```
[section]-[style]-[source].png

Examples:
hero-centered-stripe.png
pricing-cards-linear.png
testimonials-grid-notion.png
dashboard-dark-vercel.png
```
