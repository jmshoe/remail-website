# Create New Blog Post

Create a new SEO-optimized blog post for the REmail website.

## Arguments
- $ARGUMENTS: The topic or title for the blog post

## Instructions

1. Research the topic using web search to understand:
   - Current trends and statistics
   - Competitor content gaps
   - Target keywords and search intent

2. Create the blog post file at `content/blog/[slug].mdx` with:
   - SEO-optimized title (under 60 characters)
   - Meta description (under 160 characters)
   - Target primary and secondary keywords
   - Proper heading structure
   - Internal links to related content
   - Call-to-action for REmail signup

3. Content requirements:
   - Minimum 1,500 words for comprehensive coverage
   - Include statistics and data points
   - Add FAQ section with schema markup potential
   - Use conversational tone for AI search optimization
   - Include practical examples for real estate investors

4. Frontmatter template:
```yaml
---
title: ""
description: ""
publishedAt: ""
author: "REmail Team"
category: ""
tags: []
image: "/images/blog/"
imageAlt: ""
---
```

5. After creating, suggest related internal links and any images needed.
