# Competitor Analysis

Analyze competitors in the real estate direct mail space.

## Arguments
- $ARGUMENTS: Competitor name or URL to analyze (optional, defaults to main competitors)

## Main Competitors to Track
- Ballpoint Marketing
- REI Print Mail
- Open Letter Marketing
- Yellow Letter HQ
- REsimpli (direct mail feature)
- REI BlackBook (direct mail feature)

## Analysis Areas

### 1. Product Features
- What direct mail products do they offer?
- Pricing structure
- Automation capabilities
- CRM integrations
- List building/skip tracing features

### 2. SEO & Content Strategy
- What keywords are they ranking for?
- Blog content topics
- Backlink profile insights
- Content gaps we can exploit

### 3. Marketing Approach
- Value propositions used
- Target audience messaging
- Social proof (case studies, testimonials)
- Lead magnets offered

### 4. Technical Implementation
- Website speed/performance
- User experience highlights
- Mobile optimization

## Output Format
Provide findings in a structured report with:
- Competitor strengths
- Competitor weaknesses
- Opportunities for REmail
- Recommended action items

## Update Frequency
This analysis should be run monthly to track changes.
